Detailed Description

Overlays the Websocket protocol over a QTcpSocket.

See also
TorcHTTPServer
TorcHTTPRequest
TorcWebSocketThread

TorcWebSocket is NOT thread safe. It will make direct function calls to registered services - which must ensure those functions are thread safe. Likewise RemoteRequest etc can be called from any thread. For thread safe operation - use TorcWebSocketThread.

Note
SubProtocol support is currently limited to JSON-RPC. New subprotocols with mixed frame support (Binary and Text) are not currently supported.
To test using the Autobahn python test suite, configure the suite to request a connection using 'echo' as the method (e.g. 'http://your-ip-address:your-port/echo').
